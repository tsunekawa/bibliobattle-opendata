Bibliobattle Open Data
======================

ビブリオバトルに関するメタデータを試験的に作成しています。

## ファイル

* books\_movies.tsv : プレゼン動画(Youtube ID)中で紹介されている書籍(ISBN)をリンクするためのデータ
